# Render + actor-display audit

Generated by `MegatonHammer --renderlevel [outDir]` (default `out\renders`). It spins up an
off-screen GLViewport — the **same GL pipeline the editor's 3D view uses** — and for the Test Temple
project plus an assortment of vanilla OoT/MM scenes it writes **isometric + top-down PNG renders**
(1600×1600) and audits how every placed actor displays.

## Renders produced

Test Temple + 6 OoT scenes (Deku Tree, Forest Temple, Water Temple, Jabu-Jabu, Hyrule Field, Kokiri
Forest) + 6 MM scenes (East Clock Town, Stone Tower, Great Bay, Stock Pot Inn, Woodfall, Snowhead).
Each gets `<name>_iso.png` and `<name>_top.png`. **All 13 scenes rendered iso+top successfully.**

The vanilla renders come out fully textured + shaded with real actor models; the Test Temple (a
synthetic box dungeon with dev textures) reads as shaded boxes with actor markers, as expected.

## Findings / action

1. **Zero obsolete actors** across every sampled scene — every placed actor resolves to a known
   editor entity (consistent with `--actoraudit`'s 100 % name coverage).
2. **Every actor displays** — as a model, a billboard, or at minimum a locator marker. Nothing is
   invisible.
3. **Incongruity — MM unmodeled actors get no item-icon billboard.** `ItemIconSource` reads OoT's
   `icon_item_static` (dmadata file 8) specifically, so for MM it reports unavailable and unmodeled
   MM actors fall back to the locator cross (the "marker-only" column). OoT unmodeled actors get
   item-icon sprites. They still display (the cross locates them, and this matches Valve Hammer's
   "point entity with no model → bounding-box marker" behavior), but item-icon parity for MM is a
   follow-up (needs MM's icon table located + decoded; MM's layout differs and includes masks).
4. **Geometry texture coverage 39–100 %.** The untextured remainder is overwhelmingly legitimately
   vertex-shaded geometry (untextured Gouraud polygons the game itself draws without a texture),
   not a decode failure — the fully-textured dungeons (Water Temple 99 %, Woodfall/Snowhead 100 %)
   confirm the texture pipeline is sound; lower-coverage scenes (Stock Pot Inn 39 %, Jabu 47 %) are
   interiors/organic areas with large flat-shaded surfaces.
